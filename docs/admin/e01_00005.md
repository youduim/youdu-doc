## 说明

- 消息审计的所有数据存储在MongoDB数据库。
- 消息审计插件独立运行，迁移时不影响服务端、客户端的使用，您可以自行安排时间进行迁移。
- 一般情况下，消息审计不会过于占用您的服务器资源，如果您评估后仍需迁移，可以参考此文档操作。
- 如有疑问可以提交工单或联系我们。
- 我们可以协助管理员进行迁移，优先支持付费客户。
