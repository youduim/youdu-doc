## 1. 说明

​		如果您的服务器无法访问外网，消息推送功能将无法使用。您可以将消息推送服务单独部署，例如可以部署至DMZ区域。

​		一般情况下，推送服务可以和代理服务器一起部署使用，配置方法相同，如有疑问请联系我们。

## 2. 防火墙策略

- main_ip：有度主服务器地址。
- push_ip：推送服务器地址。
