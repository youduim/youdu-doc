# 消息回调

- 内容格式

```
{
  "toBuin": $buin,
  "toApp": "$app_id",
  "encrypt": "encrypt_msg"
}
```

| 参数    | 必须 | 说明                       |
| ------- | ---- | -------------------------- |
| toBuin  | 是   | 企业号                     |
| toApp   | 是   | 应用ID                     |
| encrypt | 否   | 响应结果Json加密后的消息体 |

- 消息回调应答格式

```
{
    "errcode": 0,
    "errmsg": "ok",
    "encrypt": "$encrypt_msg"
}
```

| 参数    | 必须 | 说明                         |
| ------- | ---- | ---------------------------- |
| errcode | 是   | 全局返回码                   |
| errmsg  | 是   | 结果说明                     |
| encrypt | 否   | 加密后的消息体，需要时返回。 |

NOTE: 如果没有收到响应结果，有度服务将在24小时内定时重复请求。
