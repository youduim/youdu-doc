# 消息类型和数据格式

## 文本消息

```
{
    "sessionId": "$session_id",
    "receiver": "$receiver",
    "sender": "$sender_account",
    "msgType": "text",
    "text": {
         "content": "it is a text"
    }
}
```

参数说明：

| 参数      | 必须 | 说明                       |
| --------- | ---- | -------------------------- |
| sessionId | 否   | 发送给多人会话时填写       |
| receiver  | 否   | 发送给单人会话时填写       |
| sender    | 是   | 消息发送者账号             |
| msgType   | 是   | 消息类型。这里固定为：text |
| text      | 是   | 消息信息                   |
| content   | 是   | 消息内容，支持表情         |
