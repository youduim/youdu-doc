# 创建群

**请求说明**
a、 POST方式请求URL:
http://[SERVER:7080]/cgi/group/create?accessToken=$access_token

b、请求结构体

```
{
     "name": "$group_name",
}
```

**参数说明**

| 参数 | 必须 | 说明                                           |
| ---- | ---- | ---------------------------------------------- |
| name | 是   | 群名称。不能超过32个字符（包括汉字和英文字母） |

**返回结果**

```
{
     "id":  $group_id
}
```

id为唯一的群标示，字符串类型
