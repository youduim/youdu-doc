# 设置用户头像

**请求说明**
POST方式请求URL:
http://[SERVER:7080]/cgi/avatar/set?accessToken=$access_token&userId=$user_id

**参数说明**

| 参数   | 必须 | 说明                                                         |
| ------ | ---- | ------------------------------------------------------------ |
| userId | 是   | 用户id                                                       |
| file   | 是   | 头像文件。采用multipart-form方式上传，以file作为表单名，加密后上传，加密方式详见加解密说明 |

**返回结果**

> 全局状态码
