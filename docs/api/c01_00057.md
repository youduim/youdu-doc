# 查看群信息

**请求说明**
a、 GET方式请求URL:
http://[SERVER:7080]/cgi/group/info?accessToken=$access_token&id=$groupId
**参数说明**

| 参数 | 必须 | 说明 |
| ---- | ---- | ---- |
| id   | 是   | 群Id |

**返回结果 (encrypt)**
{

```
      "id":$groupId,
      "name":$name,
      "members":[
           {"account":$userId1, "name": $name1, "mobile": $mobile1},
           {"account":$userId2, "name": $name2, "mobile": $mobile2}
      ]
}
```
