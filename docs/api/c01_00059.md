# 添加群成员

**请求说明**
a、 POST方式请求URL:
http://[SERVER:7080]/cgi/group/addmember?accessToken=$access_token

b、请求结构体

```
{
     "id": "$group_id",
     "userList":[ "$userId1", "$userId2"]
}
```

**参数说明**

| 参数     | 必须 | 说明     |
| -------- | ---- | -------- |
| id       | 是   | 群Id     |
| userList | 是   | 成员列表 |

**返回结果**

> 全局状态码
