# 群列表

**请求说明**
a、 GET方式请求URL:
http://[SERVER:7080]/cgi/group/list?accessToken=$access_token&userId=$userId
**参数说明**

| 参数   | 必须 | 说明                          |
| ------ | ---- | ----------------------------- |
| userId | 否   | 用户id,不带参数时查询所有的群 |

**返回结果(encrypt)**

```
{
 "groupList":[
     {"id":$id,"name":$name},
     {"id":$id,"name":$name}
 ]
}
```
