# 全局状态码

### 基础错误码

| 错误码                  | 十进制数值 | 含义                    |
| ----------------------- | ---------- | ----------------------- |
| ERRCODE_OK              | 0          | 操作成功,查询结果逻辑真 |
| ERRCODE_INTERNALERR     | -1         | 异常,未定义的系统错误   |
| ERRCODE_FALSE           | 2          | 查询结果逻辑假          |
| ERRCODE_NORIGHT         | 3          | 接口权限不足            |
| ERRCODE_ILLEGAL_ACCESS  | 4          | 非法访问数据            |
| ERRCODE_INVALID_REQUEST | 5          | 错误的请求              |
