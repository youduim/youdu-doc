# 删除部门

**请求说明**
GET方式请求URL:
http://[SERVER:7080]/cgi/dept/delete?accessToken=$access_token&id=$dept_id

**参数说明**

| 参数 | 必须 | 说明                                                         |
| ---- | ---- | ------------------------------------------------------------ |
| id   | 是   | 部门id，整型。不能删除根部门；不能删除含有子部门和成员的部门 |

**返回结果**

> 全局状态码
