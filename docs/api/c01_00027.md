# 第三方接口处理响应

有度会带上token和file_id跳转到后台配置的URL地址，并显示URL返回的内容。

1. 有度跳转URL，并带上参数token和file_id
2. 校验token
3. 通过SDK根据下载file_id下载zip压缩包
4. 返回html响应

返回HTML代码示例：

```
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>
        html body
    </body>
</html>
```
