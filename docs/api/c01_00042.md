# 批量删除用户

**请求说明**
a、POST方式请求URL:
http://[SERVER:7080]/cgi/user/batchdelete?accessToken=$access_token

b、请求结构体

```
{
   "delList":["$userid_1", "$userid_2", "$userid_3"]
}
```

**参数说明**

| 参数    | 必须 | 说明                          |
| ------- | ---- | ----------------------------- |
| delList | 是   | 删除的帐号列表。最多支持200个 |

**返回结果**

> 全局状态码
